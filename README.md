# Погодный парсер для Санкт-Петербурга

Этот скрипт парсит данные о погоде с сайта [world-weather.ru](https://world-weather.ru/pogoda/russia/saint_petersburg/) и сохраняет их в формате JSON.

## Что делает скрипт?

- Загружает HTML-страницу с прогнозом погоды.
- Парсит данные по дням и времени суток (ночь, утро, день, вечер).
- Извлекает следующую информацию:
  - Температура
  - Ощущается как
  - Давление
  - Влажность
  - Скорость ветра
  - Вероятность осадков
- Сохраняет результат в файл `weather.json`.
